# 🧠 Философский Магазин Идей

> _"Мы продаем не товары. Мы продаем способы видеть мир."_

Добро пожаловать в место, где валютой служит ваша **Карма**, а товаром — чистые идеи величайших умов человечества. Здесь вы не найдете физических объектов. Только сущности. Только концепции. Только то, что может изменить ваше сознание.

---

## 🌑 О Проекте

Это не просто интернет-магазин. Это интерактивная инсталляция, где каждая карточка — это портал в мышление конкретного философа. 

**Как это работает?**
1. Вы регистрируетесь (вводите уровень своей Кармы).
2. Вам открывается доступ к витрине идей.
3. У каждой идеи есть "вес" — сколько кармы она требует для усвоения.
4. Выберите идеи, переверните их, прочтите суть теории.
5. Но будьте осторожны: ваша Карма ограничена. Выбор определяет вашу реальность.

### 🎭 Что внутри?

*   **Платон**: Узнайте, почему мир вокруг вас — лишь тень.
*   **Декарт**: Проверьте, существуете ли вы на самом деле.
*   **Ницше**: Рискните заглянуть в бездну.
*   **Сартр**: Осознайте бремя собственной свободы.
*   ...и другие архитекторы человеческой мысли.

## ⚡ Технические Особенности

Этот проект — демонстрация современного фронтенда без лишних зависимостей. Чистый код, максимальная производительность и внимание к деталям.

*   🎨 **БЭМ-методология**: Строгая структура классов для масштабируемости.
*   🌗 **Тёмная тема**: Полная поддержка `dark mode` (автоматическая и ручная).
*   📱 **Адаптивность**: Идеальное отображение на любом устройстве (от телефона до ультраширокого монитора).
*   ✨ **3D Анимации**: Плавный переворот карточек с сохранением глубины сцены.
*   💎 **Sass (SCSS)**: Модульная архитектура стилей с использованием переменных и миксинов.
*   🚀 **Vanilla JS**: Никаких тяжелых фреймворков. Только нативный JavaScript.

## 🚀 Быстрый старт

Вам не нужно устанавливать сложные зависимости. Проект готов к работе сразу.

### Вариант 1: Просто открыть
Найдите файл `index.html` в корне проекта и откройте его в любом современном браузере. Всё уже работает.

### Вариант 2: Для разработчиков (с компиляцией SCSS)
Если вы хотите изменить стили:

1.  Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите процесс сборки стилей:
    ```bash
    npm run watch:css
    ```
    Теперь любые изменения в `styles.scss` будут автоматически применяться.

## 🕹️ Как пользоваться

1.  **Введите Карму**: При первом запуске система спросит ваш уровень духовного развития (число от 1 до 20). От этого зависит количество идей, которые вы сможете "приобрести".
2.  **Изучайте витрину**: Наведите курсор, чтобы увидеть легкое свечение.
3.  **Переверните карточку**: Кликните, чтобы узнать имя философа и суть его учения.
4.  **Сделайте выбор**: Нажмите кнопку "Принять идею". Если цена идеи выше вашей остаточной Кармы, карточка предостерегающе задрожит.

## 💡 Почему это необычно?

Здесь нет корзины покупок в привычном понимании. Нет доставки. Нет чеков. 
Есть только вы, экран и выбор: принять чужую мысль как свою или пройти мимо.

Интерфейс реагирует на ваши действия:
*   Недостаточно кармы? Элемент вибрирует отказом.
*   Тема меняется сама, если ваша система предпочитает темную сторону.
*   Карточки имеют "вес" и текстуру старой бумаги благодаря генеративному шуму.
